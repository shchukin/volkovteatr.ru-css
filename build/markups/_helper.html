.helper

Строка вспомогательного текста. Обычно используется с .input:

Can’t be changed later

Однако это не часть инпута. Компоненты идут друг за другом. Это делает .helper более универсальным. Его можно использовать с любым другим элементом:

Random element
Изменения сохраняются автоматически



.helper--error

Можно использовать для вывода ошибок. Можно выводить несколько штук подряд:

Password is too short
Password must contain a special character
Password must be hard to guess: dates, phone numbers and names are not allowed and either it shouldn’t be based on public data
New password is similar to previous password
