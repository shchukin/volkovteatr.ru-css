.wysiwyg

This component sets styles for tags such <p>, <ul>, <img>, etc.

On one hand, we should never match on tags globally or apply styles to tags for the sake of modularity. We always use classes. In other words it's a bad practise to apply styles directly to tags.

On the other hand, there are cases, such a user generated content or user agreement styling where we donâ€™t control the HTML code.

This is where we wrap text in a .wysiwyg class, and all the tags inside inherit the defined styles.
